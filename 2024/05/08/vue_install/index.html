vue学习之路（一）:搭建vue项目

发表于 2024-05-08 | 分类于 vue | 阅读次数

前言

Vue.js（读音 /vjuː/, 类似于 view） 是一套构建用户界面的渐进式框架。
Vue 只关注视图层， 采用自底向上增量开发的设计。
Vue 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。
Vue 学习起来非常简单。

安装nodejs

网上很多教程，也很简单，官网下载下来一直点“下一步”就行了

安装npm

nodejs默认会集成，运行命令npm -v查看是否输出版本信息，如果没有，需要手动安装

安装vuejs

安装淘宝cnpm(国内仓库，提高下载速度)

npm install -g cnpm –registry=https://registry.npm.taobao.org

安装vue-cli

cnpm install -g vue-cli

安装vue-router

cnpm install -g vue-router

使用vue-cli初始化项目

vue init webpack test-web

进入到目录

cd test-web

安装依赖

cnpm install

开始运行

npm run dev

到此，一个vuejs项目搭建起来
